Debug Window



The debugger displays six windows and a toolbar for quick access to debugging 
functions like, step, step into, step over, enabling/disabling breakpoints.  
Check out the animate feature which allows for an animated step through your code.  
Click on the led display button to toggle the display of the led box.  


Additional help and accelerator shortcuts can be obtained by moving the cursor 
on each of the toolbar button.


Modify all data and instructions displayed by the debugger by merely double clicking 
on the appropriate window.  Double click on any instruction and modify it or enter
a completely new instruction in mnemonic form!!! right in the middle of a debug session!!!.

 
Modify registers and values in memory at your will( only exception are the ports 
and stack which can be modified only via a program ) and see the changes take 
affect immediately.  All changes are highlighted in red.  Use the '+' and '-' keys
to increment/decrement/toggle the data in the various windows.
